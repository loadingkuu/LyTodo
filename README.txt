LyTodo v8

运行：
1) pip install PySide6
2) cd todo_app
3) python app.py

本版改动
- 顶部标签高亮：改成“纯直线下划线”（不会两端弯）
- 任务行：去掉标签页的小标识（只保留左侧彩色条）
- 标签栏最右：新增 🗑 按钮，一键打开“已完成收集箱浮窗”
  - 支持：恢复选中 / 删除选中 / 全部清空
- 数据结构升级为可同步：Task 增加 id(UUID)、updated_at、completed_at、order、deleted
  - 自动兼容旧 storage.json 并迁移生成必要字段


v9 更新：
- 🗑收集箱升级为：已完成 + 已删除（删除的任务会进入已删除页，可恢复或彻底删除）
- 顶部编辑框加高，并会随窗口高度自动变更（更适合长文本）
- 已附带：方案A 同步服务端/客户端脚本（server_fastapi.py / sync_client.py / README_sync.md）


v10 更新：
- 🗑按钮不再打开独立窗口：改为“已完成”标签页（主界面内切换），并在顶部出现已完成操作条（恢复/删除/清空/返回）
- 同步已接入主程序（方案A）：启动自动 pull，退出自动 push；可选后台每60秒 push
- 设置新增【同步】标签页（URL/Token/用户ID/定时推送）


v10.4 更新：
- 修复同步定时回调缺失导致启动报错
- 同步策略B：编辑/新增/删除/移动/排序等变更后 3 秒自动 push（防抖）
